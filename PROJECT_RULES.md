# School Companion Development Rules

## General

- Never rewrite existing functionality.
- Keep all components modular.
- Use TypeScript everywhere.
- Use Tailwind CSS.
- Use reusable components.
- Keep files small.
- Follow Next.js App Router conventions.
- Don't duplicate code.

## UI

- Material Design 3 inspired.
- Dark AMOLED theme.
- Rounded corners.
- Smooth animations.
- Responsive.
- Mobile-first.

## Project Structure

Components belong inside:
- components/layout
- components/navigation
- components/cards
- components/ui

Pages belong inside:
- src/app

Utilities:
- lib
- utils
- services

## Coding

- Explain new files.
- Keep comments meaningful.
- Don't remove existing code unless asked.

## Development

Implement only the requested feature.
Never implement future features automatically.
